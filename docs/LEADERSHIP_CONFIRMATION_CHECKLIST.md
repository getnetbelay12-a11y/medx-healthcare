# Leadership Confirmation Checklist

Use this checklist before publishing any current MedX director, executive, or governance profile.

| Field | Confirmation required |
| --- | --- |
| Name | Confirm spelling and preferred public name. |
| Current role | Confirm exact MedX role and effective date. |
| Credentials | Confirm degrees, licenses, and professional designations. |
| Approval to publish | Obtain written approval for website publication. |
| Photo approval | Confirm image ownership and permission for public website use. |
| Biography approval | Approve the final short biography text. |
| Profile link approval | Confirm any external profile URL before publishing. |
| Effective date | Record the date the approved information became valid. |
