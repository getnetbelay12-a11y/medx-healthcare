# Partner Confirmation Checklist

Use this checklist before publishing any organization as a current MedX partner, client, supplier, investor, or implementation relationship.

| Field | Confirmation required |
| --- | --- |
| Organization | Confirm legal name and preferred public display name. |
| Current or historical | Confirm whether the relationship is active, historical, or source context only. |
| Relationship type | Confirm category, scope, and public wording. |
| Confirmation contact | Record the approving contact at the organization or MedX. |
| Logo approval | Confirm logo ownership, usage rights, file, color, and placement rules. |
| Naming approval | Confirm the organization name can be used publicly. |
| Public-description approval | Approve the exact website description. |
| Effective date | Record the approval date and review cycle. |
